### 하위 섹션: 긴 코드 블록

긴 코드 예제입니다:

```python
import os

def list_files(directory):
    files = []
    with os.scandir(directory) as entries:
        for entry in entries:
            if entry.is_file():
                files.append(entry.name)
    return files

print(list_files("."))
```
